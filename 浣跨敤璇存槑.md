# 🐾 萌宠养成系统 - 使用说明

## 📌 快速访问链接

### 方式一：直接访问（推荐）
```
http://localhost:3000
```

### 方式二：使用启动器
双击打开 `启动器.html` 文件，点击"启动应用"按钮即可。

### 方式三：使用启动脚本
- **macOS/Linux**: 双击运行 `start.sh` 或在终端执行 `./start.sh`
- **Windows**: 双击运行 `start.bat`

## 🔄 如何更新代码

### 更新前端代码
1. 修改 `lib/` 目录下的代码
2. 运行构建命令：
   ```bash
   flutter build web --release
   ```
3. 重启后端服务

### 更新后端代码
1. 修改 `backend/` 目录下的代码
2. 重启后端服务：
   ```bash
   ./stop.sh
   ./start.sh
   ```

## 💡 常见问题

### Q: 应用无法访问？
A: 检查后端服务是否正在运行，确保3000端口未被占用。

### Q: 数据丢失了怎么办？
A: 数据保存在 `backend/database/pet_app.db`，定期备份此文件。

### Q: 如何在其他设备访问？
A: 需要配置网络转发或部署到云服务器。

### Q: 修改代码后没有生效？
A: 前端代码需要重新构建 `flutter build web --release`，后端代码需要重启服务。
