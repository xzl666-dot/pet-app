# 模块六：价值评估中心模块（详细版）
本模块是网站的**学习数据统计与成长可视化核心载体**，承担“量化学习成果、展示等级权益、激励用户进阶”的核心职责。整体界面采用“数据可视化+极简商务风”的设计风格，主色调选用沉稳的深蓝色，搭配清晰的白色背景和浅灰色分隔线，营造出专业、直观的数据分析氛围。模块围绕“等级成长、分数统计、任务分析”三大核心功能设计，与学习任务模块深度联动，让用户清晰看到每一次学习行为带来的成长变化，整体篇幅约3000字。

## 一、模块整体设计原则
1.  **数据可视化原则**
    本模块的核心目标是将用户的学习数据转化为直观易懂的图表和指标，避免枯燥的数字罗列。所有数据均采用“图表+文字说明”的组合展示方式，例如分数变化用折线图呈现、任务完成率用饼图呈现、等级进度用进度条呈现，让用户一眼就能看出自己的学习趋势和成长轨迹。图表配色与等级挂钩，不同等级对应不同的颜色标识，增强视觉辨识度。

2.  **等级激励性原则**
    模块设计以“等级提升”为核心激励点，清晰展示各等级的权益和升级条件，让用户明确进阶目标。等级提升的难度设置遵循“循序渐进”原则，从D级到S级的分数阈值逐步提高，对应的任务奖励和权限也逐步升级，形成“完成任务→积累分数→提升等级→解锁权益→完成更高阶任务”的正向循环，持续激发用户的学习动力。

3.  **数据实时性原则**
    模块与学习任务模块采用“实时数据同步”机制，用户在学习任务模块完成任务获得分数后，本模块的分数统计、等级进度、任务分析等数据会立即更新，无需手动刷新。数据更新时配有简洁的动画反馈，例如分数增长时的数字跳动动画、等级进度条的延伸动画，让用户直观感受到自己的成长变化。

4.  **操作便捷性原则**
    模块功能聚焦于“数据查看与分析”，不设置复杂的操作流程，所有功能均支持“一键筛选”“一键导出”。用户可快速切换不同的时间维度查看数据，可一键导出学习报告分享或存档，操作步骤控制在2步以内，降低用户的使用门槛。

## 二、页面整体布局设计
页面采用固定宽度的居中布局，最大宽度1200px，左右边距自适应，确保在不同分辨率下布局稳定。整体分为五大区域，按从上到下的顺序排列，区域间采用浅灰色分隔线区分，层次分明，重点突出。

1.  **顶部导航栏（通栏设计，高度60px）**
    导航栏为固定定位，始终悬浮在页面顶部，方便用户随时返回核心模块。
    - **左侧区域**：返回按钮+模块标题，返回按钮为简约的箭头图标，尺寸24×24px，颜色#1677ff，点击后跳转至学习任务模块；模块标题为“价值评估中心”，字体大小18px，颜色#333333，加粗显示，无多余装饰。
    - **右侧区域**：两个功能按钮，分别是“导出学习报告”和“返回首页”。“导出学习报告”按钮为蓝色背景，白色文字，搭配下载图标，点击后可将当前页面的核心数据导出为PDF格式；“返回首页”按钮为白色背景，蓝色边框和文字，点击后直接跳转至学习任务模块。两个按钮高度36px，字体大小14px，hover时颜色加深，交互反馈清晰。

2.  **当前等级概览区（宽度100%，高度200px）**
    这是页面的核心视觉区域，占据页面最显眼的位置，背景为深蓝色渐变，带有轻微的光影效果，营造出高级感。
    - **中央等级展示**：分为图标和文字两部分，等级图标为勋章样式，尺寸80×80px，不同等级对应不同的勋章颜色和设计，例如D级为灰色普通勋章，C级为铜色勋章，B级为银色勋章，A级为金色勋章，S级为钻石镶嵌勋章。图标下方显示“当前等级：B级”，字体大小24px，颜色白色，加粗显示；再下方显示“等级权益：解锁进阶任务，任务奖励提升10%”，字体大小14px，颜色#e6f7ff，简洁明了地告知用户当前等级的核心权益。
    - **等级进度条**：位于等级展示下方，宽度80%，高度20px，背景色#ffffff33，进度条填充色为与等级对应的颜色，例如B级为银色。进度条左侧显示“当前分数：350”，右侧显示“升级至A级需：150分”，字体大小14px，颜色白色。进度条会实时更新，分数增长时进度条会平滑延伸，带有轻微的动画效果。
    - **快速升级提示**：位于进度条下方，显示“完成今日进阶任务可获得50分，距离升级还差100分”，字体大小12px，颜色#e6f7ff，为用户提供明确的进阶建议。

3.  **分数统计区（宽度100%，高度300px）**
    位于等级概览区下方，背景为白色，带有轻微的圆角和阴影，分为左右两个子区域，左侧为折线图，右侧为关键指标。
    - **左侧分数变化折线图**：占据区域的70%，图表标题为“近30天分数变化趋势”，字体大小16px，颜色#333333。横轴为日期，纵轴为分数，折线颜色为深蓝色，数据点为实心圆，鼠标悬浮在数据点上时，会弹出提示框显示“2026-05-10 获得分数：30 来源：完成数学进阶任务”。图表支持切换时间维度，用户可点击右上角的“近7天/近30天/近90天”按钮，快速切换查看不同时间段的分数变化趋势。
    - **右侧关键指标卡片**：占据区域的30%，采用三行列表格布局，展示三个核心分数指标：“总累计分数：350”“本月新增分数：120”“单次最高得分：50”。每个指标均配有对应的图标，总累计分数配奖杯图标，本月新增分数配日历图标，单次最高得分配箭头图标，图标颜色为深蓝色，文字大小14px，颜色#333333，让用户快速掌握自己的分数概况。

4.  **任务完成分析区（宽度100%，高度300px）**
    位于分数统计区下方，背景为白色，同样分为左右两个子区域，左侧为饼图，右侧为任务类型详情。
    - **左侧任务完成率饼图**：占据区域的50%，图表标题为“任务类型完成占比”，字体大小16px，颜色#333333。饼图分为基础任务、进阶任务、挑战任务三个部分，不同部分对应不同的颜色，鼠标悬浮时弹出提示框显示“基础任务：完成20/25 占比40%”。图表支持点击图例切换显示，用户可点击“基础任务”图例，单独查看该类型任务的完成情况。
    - **右侧任务类型详情列表**：占据区域的50%，采用卡片式布局，展示每种任务类型的详细数据：基础任务（完成数/总数、完成率、平均得分）、进阶任务（完成数/总数、完成率、平均得分）、挑战任务（完成数/总数、完成率、平均得分）。卡片背景为#f5f7fa，带有轻微的圆角，文字大小12px，颜色#666666，数据清晰易读。

5.  **等级权益详情区（宽度100%，高度自适应）**
    位于页面最下方，背景为白色，是模块的辅助功能区域，为用户提供完整的等级权益参考。
    - **权益列表**：采用表格布局，表格分为四列：等级、升级所需分数、核心权益、任务权限。表格内容清晰列出从D级到S级的所有等级信息，例如D级（0分，基础奖励，仅可完成基础任务）、S级（1000分，奖励提升30%，可完成所有任务）。表格支持鼠标悬浮高亮，方便用户查看不同等级的权益对比。
    - **升级攻略**：位于表格下方，显示“升级攻略：1. 每日完成全部任务可获得稳定分数；2. 挑战高难度任务可获得额外加分；3. 宠物营养值满值时任务奖励提升”，字体大小12px，颜色#666666，为用户提供实用的升级建议。

## 三、核心功能与使用流程
本模块的核心功能围绕“等级管理、数据统计、报告导出”展开，所有功能流程均设计得简洁直观，让用户能快速获取自己的学习数据，明确进阶方向。

1.  **等级成长功能**
    等级成长是本模块的核心功能，等级的提升与用户的学习任务完成情况直接挂钩，等级权益清晰明确，激励用户持续进阶。
    - **等级划分规则**：
      1.  等级共分为5个层级，从低到高依次为D级、C级、B级、A级、S级，每个等级对应不同的分数阈值：D级（0-99分）、C级（100-199分）、B级（200-499分）、A级（500-799分）、S级（800分及以上）。
      2.  等级提升采用“分数累计”机制，用户完成任务获得的分数会累计叠加，达到升级阈值后自动提升等级，无需手动申请。
    - **等级权益规则**：
      1.  基础权益：等级每提升一级，任务奖励倍率提升10%，例如C级奖励倍率110%，S级奖励倍率140%。
      2.  任务权限：D级仅可完成基础任务，C级解锁进阶任务，B级解锁挑战任务，A级和S级可完成所有任务，且挑战任务的额外加分更高。
    - **等级提升流程**：
      1.  用户在学习任务模块完成任务，获得对应的分数，分数实时同步至本模块。
      2.  当累计分数达到升级阈值时，页面立即弹出“等级提升啦！”的弹窗，弹窗显示等级提升前后的勋章对比，同时提示“恭喜升级为A级！奖励倍率提升至130%，解锁挑战任务额外加分权益”。
      3.  弹窗自动关闭后，等级概览区的勋章图标、等级文字、进度条均会更新为新等级的状态，同时任务完成分析区的任务权限也会同步更新，用户可立即在学习任务模块体验新等级的权益。

2.  **数据统计与分析功能**
    数据统计与分析功能是本模块的核心价值所在，通过多维度的数据分析，让用户清晰了解自己的学习情况，发现学习优势和不足。
    - **多维度数据查看流程**：
      1.  用户进入模块后，默认查看近30天的分数变化趋势和任务完成占比。
      2.  用户点击分数统计区右上角的“近7天”按钮，折线图会快速切换为近7天的分数数据，数据点和趋势线同步更新，无需页面刷新。
      3.  用户点击任务完成率饼图的“进阶任务”图例，饼图会单独高亮显示进阶任务的完成占比，其他部分变为灰色，方便用户聚焦查看该类型任务的完成情况。
      4.  用户鼠标悬浮在任意数据点或图表区域时，均会弹出详细的提示框，展示该数据的具体来源和数值，帮助用户了解每一分的获取途径。
    - **学习情况分析功能**：
      1.  模块会根据用户的任务完成数据，自动生成简单的学习情况分析，例如“你的数学任务完成率高达90%，表现优秀！语文任务完成率仅60%，需要加强”，分析结果显示在任务完成分析区的下方，字体大小12px，颜色#ff4d4f，为用户提供针对性的学习建议。

3.  **学习报告导出功能**
    学习报告导出功能满足用户分享和存档学习成果的需求，报告内容涵盖核心数据，格式规范，便于查看。
    - **报告导出流程**：
      1.  用户点击顶部导航栏的“导出学习报告”按钮，系统开始生成PDF格式的学习报告，按钮进入加载状态，显示“正在生成报告...”。
      2.  报告生成完成后，按钮恢复为“导出学习报告”，同时页面弹出“报告导出成功！已保存至本地”的提示框，报告自动下载至用户的电脑。
      3.  报告内容包含：用户基本信息（昵称、等级）、核心数据（总累计分数、等级进度、近30天分数变化趋势图）、任务完成情况（各类型任务完成率、平均得分）、等级权益和升级建议，格式清晰，排版美观。

4.  **快速升级提示功能**
    快速升级提示功能为用户提供明确的进阶路径，帮助用户高效积累分数，提升等级。
    - **提示生成规则**：
      1.  系统会根据用户当前的分数和距离下一级的分数差，自动计算完成哪些任务可以快速升级，例如“完成今日3个进阶任务可获得60分，距离升级还差40分”。
      2.  提示内容会实时更新，用户完成任务后，提示内容会同步调整，始终为用户提供最新的进阶建议。
    - **提示展示方式**：
      1.  提示内容显示在等级进度条下方，字体颜色醒目，用户进入模块后可直接看到。
      2.  用户点击提示内容中的“今日进阶任务”链接，可直接跳转至学习任务模块的进阶任务列表，快速开始学习，缩短操作路径。

## 四、跳转逻辑设计
本模块作为学习数据统计中心，与学习任务模块的跳转逻辑设计得清晰便捷，确保用户在查看数据后能快速返回任务模块，将分析结果转化为学习行动。

1.  **模块内跳转**
    - 点击分数统计区的时间维度按钮（近7天/近30天/近90天）→ 实时切换折线图数据，无页面跳转。
    - 点击任务完成率饼图的图例 → 实时切换饼图显示状态，无页面跳转。
    - 点击等级权益详情区的升级攻略链接 → 弹出详细攻略弹窗，弹窗支持点击右上角关闭按钮关闭。

2.  **跨模块跳转**
    - 点击顶部导航栏的返回按钮/返回首页按钮 → 跳转至**模块四 学习任务模块**，跳转过程中添加淡入淡出动画，时长300ms，确保页面切换流畅。
    - 点击快速升级提示中的任务链接 → 跳转至学习任务模块对应的任务列表，例如点击“今日进阶任务”链接，跳转至进阶任务列表，聚焦该类型任务。
    - 等级提升弹窗点击“立即体验”按钮 → 跳转至学习任务模块的新解锁任务列表，激励用户立即开始学习。

3.  **外部跳转**
    - 点击“导出学习报告”按钮 → 触发PDF文件下载，无页面跳转，下载完成后弹出提示框，支持点击“打开文件”查看报告。

## 五、通用交互与异常处理规则
1.  **通用交互规则**
    - 页面滚动时，顶部导航栏固定悬浮，其他区域正常滚动，确保用户随时能进行导出报告和返回首页的操作。
    - 所有图表均支持鼠标悬浮交互，数据点和图例的hover效果清晰，提示框信息完整，帮助用户深入了解数据详情。
    - 按钮和可点击元素均添加明确的hover效果，鼠标悬浮时颜色加深或出现下划线，光标变为手型，明确提示用户可点击。
    - 数据更新时配有简洁的动画反馈，例如分数增长时的数字跳动、进度条的平滑延伸，增强用户的视觉体验和成就感。

2.  **异常处理规则**
    - 网络中断时，页面弹出“网络异常，请检查后重试”的提示框，包含“重试”按钮，点击后重新加载页面数据，无需用户重新登录。
    - 数据加载失败时，分数统计区和任务完成分析区显示“数据加载失败，请点击重试”的提示文字，配有刷新图标，点击后重新发起请求。
    - 用户暂无数据时（新用户首次进入），页面显示“暂无学习数据，快去完成任务获取分数吧！”的提示文字，配有引导图标，点击图标可直接跳转至学习任务模块。
    - 报告导出失败时，页面弹出“报告导出失败，请稍后重试”的提示框，支持点击“重试”按钮重新生成报告，确保用户能成功导出自己的学习数据。
