# 模块五：动物养成模块（详细版）
本模块是学习任务模块的延伸功能载体，承担“强化用户粘性、通过宠物陪伴提升学习持续性”的核心职责。整体界面采用“萌系卡通+温馨治愈”的设计风格，主色调选用柔和的暖黄色，搭配浅粉色点缀，营造出轻松可爱的氛围。模块围绕“宠物成长、道具管理、形象解锁”三大核心功能设计，与学习任务模块深度联动，让宠物成为用户学习路上的专属陪伴者，整体篇幅约3000字。

## 一、模块整体设计原则
1.  **陪伴激励性原则**
    本模块的核心目标是通过宠物的成长与互动，激励用户持续完成学习任务。宠物的属性变化、形象解锁均与学习任务的完成情况强绑定，让用户直观感受到“学习行为”带来的“宠物成长反馈”。所有交互设计均突出“陪伴感”，例如宠物会根据用户操作做出不同的互动动作，离线后再次登录时宠物会弹出欢迎提示，强化用户与宠物之间的情感连接。

2.  **视觉萌系化原则**
    界面整体采用卡通萌系风格，宠物形象设计圆润可爱，色彩鲜明柔和，符合高中生的审美偏好。页面元素均采用圆角设计，避免尖锐棱角，降低视觉压迫感。宠物的动作动画流畅自然，例如进食、跳跃、升级等动作均设计有专属的萌系动画，时长控制在1-2秒，增强趣味性。

3.  **功能轻量化原则**
    模块功能聚焦于“宠物养成”核心，不添加冗余的复杂玩法，所有操作流程控制在2步以内。例如道具投喂仅需“选择道具→点击投喂”，道具合成仅需“选择道具→点击合成”，降低用户操作成本。页面信息展示简洁明了，核心属性（等级、经验、营养值）优先展示，次要信息（成长记录）可通过点击展开查看，避免信息过载。

4.  **数据联动性原则**
    模块与学习任务模块实时数据互通，宠物的经验值、营养值、饲料数量等数据均来自学习任务的奖励，宠物状态反过来影响学习任务的奖励倍率。数据同步采用“双向实时更新”机制，用户在本模块投喂饲料提升营养值后，学习任务模块的奖励倍率会立即调整；用户在学习任务模块获得经验值后，本模块的宠物经验值进度条会实时增长。

## 二、页面整体布局设计
页面采用固定宽度的居中布局，最大宽度1100px，左右边距自适应，确保在不同分辨率下布局稳定。整体分为四大区域，按从上到下的顺序排列，区域间采用可爱的卡通分隔线区分，增强萌系氛围。

1.  **顶部导航栏（通栏设计，高度55px）**
    导航栏为固定定位，始终悬浮在页面顶部，方便用户随时返回核心模块。
    - **左侧区域**：返回按钮+模块标题，返回按钮为卡通箭头样式，尺寸24×24px，点击后跳转至学习任务模块；模块标题为“我的专属宠物”，字体大小18px，颜色#e6a23c，加粗显示，带有轻微的阴影效果。
    - **右侧区域**：两个功能按钮，分别是“成长记录”和“返回首页”。“成长记录”按钮为卡通日历样式，点击后弹出宠物成长记录弹窗；“返回首页”按钮为卡通房子样式，点击后直接跳转至学习任务模块。两个按钮均采用萌系设计，尺寸36×36px，鼠标悬浮时会轻微放大，增强交互趣味性。

2.  **宠物形象展示区（宽度100%，高度400px）**
    这是页面的核心视觉区域，占据页面最大空间，背景为蓝天白云的卡通图案，营造出轻松愉悦的氛围。
    - **中央宠物动态形象**：宠物形象根据等级不同呈现不同的形态，例如LV.1为迷你幼崽形态，LV.5为成长形态，LV.10为成熟形态，等级越高形象越精致可爱。宠物形象尺寸随等级提升而增大，LV.1尺寸为120×120px，LV.10尺寸为220×220px。宠物会根据自身状态做出不同的动态交互，例如营养值充足时会欢快跳跃、转圈，营养值不足时会低头耷拉耳朵，用户点击宠物时会做出撒娇、蹭手等互动动作。
    - **等级与经验展示**：位于宠物形象下方，分为两部分。一是等级标识，显示“宠物等级 LV.8”，字体大小22px，颜色#e6a23c，加粗显示，带有金色描边；二是经验进度条，宽度300px，高度20px，进度条背景为浅灰色，填充色为暖黄色，进度条右侧显示“经验值 240/300”，字体大小14px，颜色#666666。进度条会实时更新，当经验值达到升级阈值时，会出现闪烁的星光动画提示。

3.  **宠物属性面板区（宽度100%，高度180px）**
    位于宠物形象展示区下方，采用四宫格卡片式布局，每个卡片宽度220px，高度150px，带有轻微的阴影和圆角，背景为白色。四个卡片分别对应宠物的核心属性，按从左到右的顺序排列：
    - **营养值卡片**：图标为卡通餐盘样式，显示“营养值 92/100”，下方配有进度条，进度条颜色为粉色。卡片下方标注“影响学习任务奖励倍率”，字体大小12px，颜色#999999。
    - **亲密度卡片**：图标为卡通爱心样式，显示“亲密度 85/100”，下方配有进度条，进度条颜色为红色。卡片下方标注“亲密度满值后升级经验减少10%”，字体大小12px，颜色#999999。
    - **体力值卡片**：图标为卡通闪电样式，显示“体力值 10/10”，下方配有进度条，进度条颜色为蓝色。体力值影响宠物互动次数，每点击宠物互动一次消耗1点体力值，每日凌晨自动恢复满值。
    - **拥有道具卡片**：图标为卡通背包样式，显示“饲料×25 亲密度道具×8”，点击卡片可直接跳转至道具背包区，方便用户快速查看道具数量。

4.  **核心操作区（宽度100%，高度220px）**
    位于页面最下方，分为左右两个子区域，左侧为道具投喂区，右侧为道具背包与合成区，两个区域之间采用卡通栅栏分隔线区分。
    - **左侧道具投喂区（宽度500px）**：包含“投喂饲料”和“使用亲密度道具”两个核心按钮，按钮宽度200px，高度50px，采用萌系设计，背景色分别为暖黄色和粉色，文字颜色为白色，字体大小16px。按钮下方标注使用规则，“投喂饲料：每次消耗1个，营养值+5”“亲密度道具：每次消耗1个，亲密度+3”。按钮禁用条件为对应道具数量为0，禁用时背景色为浅灰色，无法点击。
    - **右侧道具背包与合成区（宽度550px）**：分为上下两部分，上半部分为道具背包列表，采用横向滚动布局，显示用户拥有的所有道具，每个道具均为卡通图标样式，尺寸50×50px，图标下方标注道具名称和数量；下半部分为道具合成按钮，按钮宽度180px，高度40px，背景色为绿色，文字颜色为白色，显示“道具合成（3合1）”。点击合成按钮后，弹出合成选择弹窗，用户可选择3个相同的低级道具合成1个高级道具，高级道具的效果是低级道具的2倍。

## 三、核心功能与使用流程
本模块的核心功能围绕“宠物成长、道具管理、互动陪伴”展开，所有功能流程均设计得简单易懂，让用户能快速上手，享受宠物养成的乐趣。

1.  **宠物成长功能**
    宠物成长是本模块的核心功能，成长所需的经验值完全来自学习任务模块的奖励，成长过程伴随形象解锁和属性提升，形成正向激励。
    - **经验值获取与升级流程**：
      1.  用户在学习任务模块完成任务后，获得的宠物经验值会实时同步至本模块，经验值进度条随之增长。
      2.  当经验值达到当前等级的升级阈值时，页面立即弹出“宠物升级啦！”的萌系弹窗，弹窗显示宠物升级前后的形象对比图，同时提示“恭喜LV.7→LV.8！解锁新形象，体力值上限+2”。
      3.  弹窗自动关闭后，宠物形象更新为新等级的形态，同时触发升级庆祝动画，宠物周围出现闪烁的星光和彩带，持续1.5秒。
      4.  升级后宠物的属性上限提升，例如体力值上限从8点提升至10点，亲密度满值后升级经验减少的比例保持10%不变。
    - **形象解锁规则**：宠物形象每5级解锁一次新形态，例如LV.1、LV.5、LV.10分别对应幼崽、成长、成熟三种形态。解锁新形态时，除了弹窗提示外，还会在成长记录中自动存档，用户可随时查看形象变化历程。

2.  **道具管理与使用功能**
    道具是宠物成长的重要辅助，分为饲料、亲密度道具两类，均来自学习任务的奖励，支持投喂和合成，提升养成效率。
    - **道具投喂流程**：
      1.  饲料投喂：用户点击“投喂饲料”按钮，系统首先校验饲料数量是否大于0，若数量充足则消耗1个饲料，宠物营养值+5，同时触发宠物进食动画，宠物会做出低头吃饲料的可爱动作，页面弹出“投喂成功！营养值+5，当前营养值92/100”的提示框，1秒后自动消失。
      2.  亲密度道具使用：用户点击“使用亲密度道具”按钮，系统校验道具数量是否大于0，若数量充足则消耗1个道具，宠物亲密度+3，触发宠物撒娇互动动画，页面弹出“使用成功！亲密度+3，当前亲密度85/100”的提示框。
    - **道具合成流程**：
      1.  用户点击“道具合成（3合1）”按钮，弹出合成选择弹窗，弹窗中显示所有可合成的道具类型，例如“低级饲料（3个→高级饲料）”“低级亲密度道具（3个→高级亲密度道具）”。
      2.  用户选择需要合成的道具类型，系统校验该道具数量是否≥3，若满足条件则消耗3个低级道具，生成1个高级道具，高级饲料投喂后营养值+10，高级亲密度道具使用后亲密度+6。
      3.  合成成功后，弹窗显示“合成成功！获得高级饲料×1”，同时道具背包列表实时更新道具数量，弹窗支持点击“确定”按钮关闭。

3.  **宠物互动陪伴功能**
    互动陪伴功能是提升用户粘性的关键，通过简单的点击交互，让宠物成为有温度的学习伙伴。
    - **日常互动流程**：
      1.  用户点击宠物形象，系统首先校验体力值是否大于0，若体力值充足则消耗1点体力值，宠物做出对应的互动动作，例如蹭手、转圈、撒娇等，不同动作随机触发。
      2.  每次互动后，有10%的概率触发“惊喜奖励”，奖励1个低级饲料，页面弹出“宠物很开心！奖励饲料×1”的提示框，增强互动的趣味性。
      3.  当体力值消耗至0时，宠物会做出疲惫的动作，此时点击宠物不再触发互动，页面弹出“宠物累啦，明天再来互动吧”的提示框。
    - **离线回归欢迎功能**：用户离线超过24小时后再次登录本模块，宠物会立即弹出欢迎弹窗，显示“欢迎回来！我好想你呀~”，同时触发宠物跳跃欢迎的动画，让用户感受到专属的陪伴感。

4.  **成长记录查询功能**
    成长记录功能让用户可以回顾宠物的成长历程，增强成就感和归属感。
    - **记录查询流程**：
      1.  用户点击顶部导航栏的“成长记录”按钮，弹出成长记录弹窗，弹窗按时间倒序排列宠物的关键成长节点，例如“2026-05-10 宠物升级至LV.8”“2026-05-05 解锁成长形态”“2026-05-01 获得第一份饲料奖励”。
      2.  每条记录均配有对应的卡通图标，升级记录配有升级图标，形象解锁记录配有形象图标，奖励记录配有道具图标，方便用户快速识别。
      3.  弹窗支持“导出记录”功能，用户点击导出按钮后，可将成长记录导出为图片格式，分享至社交平台，展示自己的宠物成长历程。

## 四、跳转逻辑设计
本模块作为学习任务模块的延伸，跳转逻辑设计得清晰便捷，确保用户在两个模块间切换流畅，不中断体验。

1.  **模块内跳转**
    - 点击拥有道具卡片 → 跳转至道具背包与合成区，自动滚动至对应位置，无页面跳转。
    - 点击道具合成按钮 → 弹出合成选择弹窗，弹窗支持点击“关闭”按钮或空白处关闭。
    - 点击成长记录按钮 → 弹出成长记录弹窗，弹窗支持点击“导出”“关闭”按钮操作。

2.  **跨模块跳转**
    - 点击顶部导航栏的返回按钮/返回首页按钮 → 跳转至**模块四 学习任务模块**，跳转过程中添加萌系过渡动画，宠物形象会短暂出现在页面中间，然后消失，增强趣味性。
    - 宠物升级解锁新形态后，弹窗底部会显示“返回任务页继续学习”按钮 → 点击后跳转至学习任务模块，激励用户继续完成任务。

3.  **外部跳转**
    - 点击成长记录弹窗的“导出记录”按钮 → 触发图片下载，无页面跳转，下载完成后弹出“记录导出成功”的提示框。

## 五、通用交互与异常处理规则
1.  **通用交互规则**
    - 页面滚动时，顶部导航栏固定悬浮，其他区域正常滚动，确保用户随时能返回核心模块。
    - 所有按钮、道具图标均添加hover效果，鼠标悬浮时轻微放大并变色，光标变为手型，明确提示可点击。
    - 宠物的所有动作动画均设计有流畅的过渡效果，避免卡顿，增强萌系氛围。
    - 页面支持“一键隐藏操作区”功能，用户点击宠物形象展示区的空白处，可隐藏核心操作区，专注欣赏宠物形象，再次点击则显示操作区。

2.  **异常处理规则**
    - 网络中断时，页面弹出萌系提示框，显示“网络开小差啦，点击重试吧”，提示框配有卡通哭脸图标，包含“重试”按钮，点击后重新加载页面数据，无需用户重新登录。
    - 道具数量不足时，点击投喂或合成按钮，页面弹出提示框，显示“道具数量不足，快去完成学习任务获取吧”，提示框配有卡通提示图标，1.5秒后自动消失。
    - 数据加载失败时，页面显示“宠物还在赶来的路上，点击重试吧”的提示文字，配有卡通加载图标，点击后重新发起请求。
    - 用户长时间未操作（超过20分钟），宠物会做出挥手提醒的动作，页面弹出“快来和我互动吧~”的提示框，增强用户的互动意愿。
